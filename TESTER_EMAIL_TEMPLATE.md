- Include all required information (see guide for details)
- Specify the severity level

### 🐛 Bug Reporting

When you find a bug, please report it with:
- **Bug Title** - Clear, concise description
- **Severity** - Critical / High / Medium / Low
- **Steps to Reproduce** - Exact steps to trigger the bug
- **Expected Behavior** - What should happen
- **Actual Behavior** - What actually happened
- **Screenshots/Video** - Visual evidence
- **Device & Browser** - Device type, OS, browser version
- **Timestamp** - When the bug occurred

**Report to:** testing@familyconnect.app

### 💡 Testing Tips

1. **Test Systematically** - Follow the checklist in order
2. **Test on Multiple Devices** - Use different devices and browsers
3. **Test Edge Cases** - Try slow networks, offline mode, large files
4. **Document Issues** - Add detailed notes for any failures
5. **Report Promptly** - Report critical bugs immediately
6. **Be Thorough** - Test all user roles and permissions
