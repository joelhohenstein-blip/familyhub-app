# Image Compression CLI Tool Guide

## Overview

The Image Compression CLI tool provides a command-line interface for batch processing, analyzing, and optimizing images using the compression profiles and quality levels defined in the architecture.

## Installation

```bash
# Install as development dependency
bun add -D image-compression-cli

# Or install globally
bun install -g image-compression-cli

# Or use directly with bunx
bunx image-compression-cli --help
```

## Quick Start

### Basic Compression

```bash
# Compress single image with mobile profile
compress-cli compress ./photo.jpg --profile mobile --output ./photo-mobile.jpg

# Compress with custom quality
compress-cli compress ./photo.jpg --quality 75 --format webp --output ./photo.webp

# Generate all profiles
compress-cli compress ./photo.jpg --all-profiles --output-dir ./compressed
```

### Batch Processing

```bash
# Compress entire directory
compress-cli batch ./images --profile mobile --output-dir ./compressed

# Batch with pattern matching
compress-cli batch ./images --pattern "*.jpg" --profile desktop --output-dir ./compressed

# Recursive batch processing
compress-cli batch ./images --recursive --profile mobile --output-dir ./compressed --parallel 4
```

### Analysis

```bash
# Analyze single image
compress-cli analyze ./photo.jpg

# Analyze with recommendations
compress-cli analyze ./photo.jpg --recommendations

# Analyze directory
compress-cli analyze ./images --recursive --export json --output report.json
```

## Command Reference

### 1. Compress Command

Compress a single image or generate multiple profiles.

```bash
compress-cli compress <input> [options]
```

#### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--profile, -p` | string | - | Compression profile (thumbnail, mobile, desktop, original) |
| `--quality, -q` | number | - | Quality level (0-100) |
| `--format, -f` | string | - | Output format (jpeg, webp, png, avif) |
| `--output, -o` | string | - | Output file path |
| `--target-size` | string | - | Target file size (e.g., "150KB") |
| `--all-profiles` | boolean | false | Generate all profiles |
| `--output-dir` | string | - | Output directory for multiple files |
| `--strip-metadata` | boolean | true | Remove EXIF/ICC/XMP data |
| `--progressive` | boolean | true | Use progressive JPEG |
| `--interlace` | boolean | false | Use interlaced PNG |
| `--verbose, -v` | boolean | false | Detailed output |
| `--dry-run` | boolean | false | Preview without writing |

#### Examples

```bash
# Compress with thumbnail profile
compress-cli compress photo.jpg --profile thumbnail --output photo-thumb.jpg

# Compress with target size
compress-cli compress photo.jpg --target-size 150KB --output photo-optimized.jpg

# Generate all profiles
compress-cli compress photo.jpg --all-profiles --output-dir ./profiles

# Compress with custom quality
compress-cli compress photo.jpg --quality 75 --format webp --output photo.webp

# Dry run to preview
compress-cli compress photo.jpg --profile mobile --dry-run --verbose
```

#### Output

```
✓ Compression complete
  Input:  photo.jpg (2.5 MB)
  Output: photo-mobile.jpg (110 KB)
  Ratio:  0.044 (95.6% savings)
  Time:   450ms
```

### 2. Batch Command

Process multiple images in a directory.

```bash
compress-cli batch <input-dir> [options]
```

#### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--profile, -p` | string | mobile | Compression profile |
| `--output-dir, -o` | string | ./compressed | Output directory |
| `--pattern` | string | *.{jpg,png,webp} | File pattern to match |
| `--recursive, -r` | boolean | false | Search subdirectories |
| `--parallel` | number | 2 | Number of parallel workers |
| `--quality, -q` | number | - | Quality level (0-100) |
| `--format, -f` | string | - | Output format |
| `--strip-metadata` | boolean | true | Remove metadata |
| `--progress` | boolean | true | Show progress bar |
| `--dry-run` | boolean | false | Preview without writing |
| `--verbose, -v` | boolean | false | Detailed output |

#### Examples

```bash
# Batch compress with mobile profile
compress-cli batch ./images --profile mobile --output-dir ./compressed

# Batch with pattern matching
compress-cli batch ./images --pattern "*.jpg" --profile desktop --output-dir ./compressed

# Recursive batch with parallel processing
compress-cli batch ./images --recursive --profile mobile --output-dir ./compressed --parallel 4

# Batch with progress and verbose output
compress-cli batch ./images --profile mobile --output-dir ./compressed --progress --verbose

# Dry run to preview
compress-cli batch ./images --profile mobile --dry-run --verbose
```
