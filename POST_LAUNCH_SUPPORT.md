## 💬 HOW TO REQUEST CHANGES

### **Tell me:**
1. **What** you want changed
2. **Why** (context helps)
3. **Where** (which page/feature)
4. **How** (if you have a vision)

### **Example Good Request:**
```
"The message board text is too small on mobile. 
Can you increase the font size from 14px to 16px? 
This makes it easier for my elderly parents to read."
```

### **Example Unclear Request:**
```
"Make it better"
```

## 🐛 REPORTING BUGS

**Include:**
1. **What happened** - What did you do?
2. **What should happen** - Expected behavior
3. **Screenshots** - Show me the issue
4. **When** - Does it always happen?
5. **Who** - Which user/family did it happen to?

**Example:**
```
User: Sarah tried to upload a photo to the gallery.
Expected: Photo appears in gallery within 2 seconds
Actual: Photo upload hangs, then shows error "File too large"
Screenshot: [attached]
Happens: Every time, only on mobile
```

## 🎯 THINGS YOU CAN EASILY CHANGE YOURSELF

**No coding needed - edit these files directly:**

1. **Colors/Theme** → `app/globals.css` or `tailwind.config.ts`
2. **Text labels** → `app/locales/en.json` (English), `es.json` (Spanish), `fr.json` (French)
3. **Logo** → Replace `/public/family-logo.svg`
4. **Fonts** → `app/globals.css`
5. **Page titles** → Update in route files
