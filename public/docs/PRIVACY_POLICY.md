## 5. Data Security

### 5.1 Security Measures

We implement industry-standard security measures:

**In Transit:**
- HTTPS/TLS encryption for all data
- Secure WebSocket (WSS) for real-time features
- Certificate pinning for mobile apps

**At Rest:**
- AES-256 encryption for sensitive data
- Encrypted database backups
- Secure key management

**Access Control:**
- Role-based access control (RBAC)
- Multi-factor authentication (MFA) available
- Principle of least privilege
- Regular access audits

**Monitoring:**
- 24/7 security monitoring
- Intrusion detection systems
- Regular security audits
- Penetration testing

### 5.2 Data Retention

We retain your data:
- **Active accounts:** As long as you use FamilyHub
- **Deleted accounts:** 30 days (for recovery), then permanently deleted
- **Backups:** Up to 90 days
- **Logs:** 30-90 days (for security and compliance)

You can request data deletion anytime. We'll delete your data within 30 days.

### 5.3 Limitations

While we implement strong security measures, no system is 100% secure. We cannot guarantee absolute security. You use FamilyHub at your own risk.

**Rectification** - Correct inaccurate data  
**Erasure** - Delete your data ("right to be forgotten")
